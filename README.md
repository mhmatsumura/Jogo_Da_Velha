# Jogo Da Velha (ANDROID)
#### Aplicativo desenvolvido no TCC da pós graduação. Foi implementado o algoritmo de APRENDIZADO POR REFORÇO Q-Learning.

##### Tecnologias utilizadas:
- BaseAdapter
- ImageView
- TextView 
- ViewGroup
- LayoutInflater
- SimpleDateFormat
- ArrayList
- List
- SQLiteDatabase
- SQLiteOpenHelper
- Serializable
- BroadcastReceiver
